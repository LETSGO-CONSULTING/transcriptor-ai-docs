# Tech Stack

## Frontend

| Technology | Version | Purpose |
| :--- | :--- | :--- |
| **Electron** | Latest | Allows creating a desktop application and accessing system audio. |
| **Angular** | v19 | Robust framework for building the user interface and managing state. |
| **Tailwind CSS** | v4 | Fast and modern interface styling. |
| **Socket.io-client** | v4 | Client for real-time communication with the backend. |

## Backend

| Technology | Version | Purpose |
| :--- | :--- | :--- |
| **NestJS** | v11 | Scalable and modular Node.js framework for the server. |
| **Prisma** | v6 | Modern ORM for interacting with the PostgreSQL database. |
| **Socket.io** | v4 | WebSockets server for real-time communication. |
| **Passport** | - | Authentication middleware (JWT, Google, Apple). |

## Infrastructure and Services

| Service | Purpose |
| :--- | :--- |
| **PostgreSQL** | Relational database for storing users and configurations. |
| **Azure Speech Services** | AI engine for voice transcription and translation. |
| **Docker** | (Optional) For containerizing the application and database. |
